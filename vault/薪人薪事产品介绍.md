# 薪人薪事产品介绍

薪人薪事是数据驱动“人效提升”的智能一体化HR SaaS服务平台，致力于帮助企业提升人力资源管理能力，激发组织员工和工作活力，全面提升人才管理效能。

## 公司简介

- 成立于2015年，总部位于北京
- 团队来自BAT、美团等知名互联网公司
- 专业数据化人力资源SaaS产品
- 以智能数据为基础，帮助企业提升HR管理水平

- [[薪酬考勤系统]]

## 产品亮点

1. 全模块一体化：HR全模块、全场景数据操作留痕，形成“人事+财务+业务”一体化管理
2. AI原生设计：机器学习、大数据等技术深度应用，简历解析、智能推荐、数据洞察全链路AI加持
3. 开放生态互联：打通各IM平台，与主流业务、财务、银行、国税平台对接，开放强大API
4. 灵活行业适配：可灵活运用到30+行业，承载300+管理方案，满足不同行业特殊需求
